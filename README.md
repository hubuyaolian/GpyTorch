# 🌟 GpyTorch: AI 进化史实战大师课
> **从感知机的诞生，到 Transformer 的觉醒之路**

本项目是一套专为 **M1 Mac (Apple Silicon)** 用户设计的深度学习进阶实战课程。我们摒弃了传统的“调包式”教学，而是以 **AI 进化史** 为主线，带你重走一遍深度学习的破局历程。

---

## 🚀 核心教学理念：步步为营，直击灵魂

1. **问题驱动**：每一章都从历史上的“卡脖子”痛点出发（例如：单层感知机为什么分不开 XOR 数据？）。
2. **技术溯源**：详细拆解每一个 AI 术语（如：反向传播、残差连接、自注意力）是为了解决什么问题而诞生的。
3. **图文并茂**：使用丰富的文字说明、数学直觉剖析和 ASCII 原理图，确保你在写代码前先看透本质。
4. **渐进式实现**：采用“学一个概念，写一段代码”的交织排版，让你在沉浸式阅读中亲手构建模型。

---

## 📅 12 周进化史课程大纲

### ❄️ 第一阶段：AI 的初春与寒冬 (Week 01-02)
*   **名词**：感知机 (Perceptron)、权重、偏置、线性分类边界、XOR 危机。
*   **目标**：理解神经网络的最底层逻辑，复现导致 AI 第一次寒冬的 XOR 危机。

### ☀️ 第二阶段：破局与复兴 (Week 03-04)
*   **名词**：隐藏层、非线性激活 (Sigmoid/ReLU)、反向传播 (Backprop)、链式法则。
*   **目标**：掌握信用分配问题的解法，手写反向传播逻辑，突破非线性鸿沟。

### 👁️ 第三阶段：视觉的征服与瓶颈 (Week 05-06)
*   **名词**：卷积核 (Kernel)、感受野、池化、权值共享、梯度消失。
*   **目标**：理解图像维度灾难，手写滑动窗口卷积，构建基础 CNN。

### 🌉 第四阶段：ResNet 的奇迹 (Week 07-08)
*   **名词**：残差连接 (Skip Connection)、深度退化问题、ResNet。
*   **目标**：理解为何“简单即是伟大”，复现极简残差块，跨越深度鸿沟。

### ⏳ 第五阶段：记忆的诞生 (Week 09-10)
*   **名词**：循环神经网络 (RNN)、隐藏状态 (Hidden State)、LSTM、门控机制。
*   **目标**：赋予机器处理序列的能力，理解上下文记忆的流转与保存。

### 🚀 第六阶段：注意力时代 (Week 11-12)
*   **名词**：自注意力机制 (Self-Attention: Q/K/V)、Transformer。
*   **目标**：彻底理解现代大模型的底层心脏，手拆矩阵乘法，透视 Attention 的本质。

## 🛠️ 环境配置指南 (针对 M1 Mac)

本项目深度优化了对 **Apple Silicon (MPS)** 的支持，可调用 M1 芯片的 GPU 加速计算。

1.  **安装环境管理器**：推荐使用 [Miniforge](https://github.com/conda-forge/miniforge)。
2.  **创建虚拟环境**：
    ```bash
    conda create -n pytorch_env python=3.10 -y
    conda activate pytorch_env
    ```
3.  **安装专为 M1 编译的 PyTorch**：
    ```bash
    pip install torch torchvision torchaudio -i https://pypi.tuna.tsinghua.edu.cn/simple
    ```

## 📖 如何开始学习？

1.  在 **Trae** (或 VSCode) 中打开本项目文件夹。
2.  进入 `Week01` 目录，点击打开 `01_感知机与XOR危机.ipynb`。
3.  **关键步骤：选择内核 (Kernel)**
    *   点击 Notebook 右上角的 "Select Kernel"。
    *   选择 "Python Environments..."。
    *   找到并选中刚才创建的 `pytorch_env`。
4.  按照文档指引，边读、边看、边运行代码。

## 🐍 编写规范
本项目所有演示代码严格遵循：
*   **符合 Flake8 规范**：保证代码整洁度。
*   **保姆级 Docstring**：说明每个类和方法的功能与参数。
*   **逐行中文注释**：不漏掉任何一行数学逻辑的解释。
*   **拒绝复杂语法**：不使用三元运算符和复杂的单行推导式，以保证最高的可读性。
