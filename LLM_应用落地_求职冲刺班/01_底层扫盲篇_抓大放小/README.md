# 🏃 01. 底层扫盲篇：抓大放小

> **学习原则：** 这一部分对应原有《AI进化史》的 Week 01 到 Week 16。对于旨在做“应用落地”的你来说，**千万别陷进去！**

## 📖 粗读清单与目标：

1. **Transformer 怎么回事？**
   * 不要求：手推 Attention 公式。
   * 要求：知道输入是 Token，输出是下一个 Token 的概率分布。理解 Q（查询）、K（键）、V（值）的通俗含义（例如图书馆找书比喻）。
2. **Embedding 是什么？（极其重要）**
   * 不要求：自己训练一个 Word2Vec。
   * 要求：深刻理解 Embedding 就是把“语义”变成了“空间中的坐标”。两个坐标越近，意思越近。这是下一章 RAG 的基石。
3. **微调 (Fine-tuning) 的常识**
   * 不要求：去搭几十张卡的集群做预训练。
   * 要求：知道什么时候该用 Prompt，什么时候该用 RAG，什么时候该用 SFT（监督微调）。

## 💡 实践作业：
调通任何一个大模型 API（DeepSeek, 通义千问, 智谱 或 OpenAI），写一个不超过 50 行代码的“终端翻译工具”。跑通它，感受一下 API 调用的快感即可进入下一章。
