# 🤖 03. Agent 实战篇：进阶落地

> **学习原则：** 对应原有《AI进化史》Week 17-20 的升级实战版。抛弃复杂的论文理论，用最脏的业务逻辑跑通 Agent 闭环。

## 🏗️ 核心实战目录：

### 1. Function Calling (工具调用) 彻底扒皮
* 剖析：不要只用 LangChain 的现成模块，手搓一次 API 的 Function Calling 格式。理解模型是怎么返回 `tool_calls` 的。

### 2. 异常处理：大厂最关心的工程细节
* 痛点：模型脑抽死循环了怎么办？调用的外部天气 API 挂了怎么办？
* 实战：
  * **容错与降级**：API 失败时的优雅兜底逻辑。
  * **循环熔断**：限制 Agent 的最大思考步数（Max Steps）。

### 3. 多轮对话与状态保持 (Memory)
* 实战：如何在工具调用和用户对话之间穿插，优雅地维护历史对话数组 (Messages History)。不要每次都把几十轮对话全塞进去（Token 浪费），实现基于摘要或滑动窗口的记忆机制。
